About

MathWiz is an exciting project inspired by my passion for mathematics and my interest in software development. This is a project I undertook as part of my course at Holberton School, where I challenged myself to create a sophisticated Java application to help users explore and visualize mathematical functions.

The idea for MathWiz arose from my desire to combine my programming skills with my passion for mathematics. I wanted to create a user-friendly, powerful application that would enable users to generate mathematical functions, evaluate them, and represent them graphically.

As a portfolio project for Holberton School, MathWiz demonstrates my software development skills, using JavaFX libraries to create an interactive and functional user interface. I chose to use JAva & JavaFX because of its rich functionality and ease of use for creating desktop applications.

Features

The application generates curves of mathematical functions and their derivatives.
It gives curves in different colors to differentiate intersections between them and their derivatives.
Of course, it performs necessary operations like addition, subtraction, division, multiplication, power, and so on.
